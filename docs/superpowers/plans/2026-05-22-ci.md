# GitHub Actions CI Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a useful GitHub Actions CI workflow for `rust-analyzer-mcp`.

**Architecture:** Create one workflow file with a Ubuntu-only quality job and a Ubuntu/Windows test matrix job. The workflow follows the checked-in design spec and the README's local development commands, with `--locked` added for reproducibility.

**Tech Stack:** GitHub Actions, Rust stable, Cargo, rustfmt, Clippy, rust-analyzer, `actions/checkout`, `dtolnay/rust-toolchain`, `Swatinem/rust-cache`.

---

### Task 1: Add CI Workflow

**Files:**
- Create: `.github/workflows/ci.yml`
- Reference: `docs/superpowers/specs/2026-05-22-ci-design.md`

- [x] **Step 1: Create the workflow directory and file**

Add `.github/workflows/ci.yml` with this exact content:

```yaml
name: CI

on:
  push:
    branches: [main]
  pull_request:
    branches: [main]

permissions:
  contents: read

jobs:
  quality:
    name: Quality
    runs-on: ubuntu-latest

    steps:
      - name: Checkout
        uses: actions/checkout@v6

      - name: Install Rust
        uses: dtolnay/rust-toolchain@stable
        with:
          components: rustfmt, clippy

      - name: Cache Cargo
        uses: Swatinem/rust-cache@v2

      - name: Check formatting
        run: cargo fmt --check

      - name: Run clippy
        run: cargo clippy --locked --all-targets --all-features -- -D warnings

  test:
    name: Test (${{ matrix.os }})
    runs-on: ${{ matrix.os }}
    strategy:
      fail-fast: false
      matrix:
        os: [ubuntu-latest, windows-latest]

    steps:
      - name: Checkout
        uses: actions/checkout@v6

      - name: Install Rust
        uses: dtolnay/rust-toolchain@stable

      - name: Install rust-analyzer
        run: rustup component add rust-analyzer

      - name: Verify rust-analyzer
        run: rust-analyzer --version

      - name: Cache Cargo
        uses: Swatinem/rust-cache@v2

      - name: Run tests
        run: cargo test --locked --all
```
